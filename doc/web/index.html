relacs is a software platform for online data acquisition, analysis, and stimulus generation specifically designed for (but not limited to) electrophysiological recordings.

Overview

Closed loop experiments.
  • Closed-loop experiments
  • Through C++ plugins relacs is very flexible and adaptable to your specific needs.
  • Integrating experiments and model simulations
  • Hardware independence
  • Multi-threaded design
  • Data-analysis library
  • Meta-data
  • Free software

We are using this program very succesfully in our lab (see list of publications).

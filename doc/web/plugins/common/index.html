Common Plugins

Plugins that can be used by any application.

  • EPhysTraces: Variables for standard input traces and events in electrophysiology.
  • HighPass: A simple first order high pass filter.
  • LowPass: A simple first order low pass filter.
  • MotorControl: Control widgets for motorized manipulators.
  • NeuronModels: Various models of spiking (point-) neurons.
  • Pause: Does nothing.
  • RestartDelay: Measures delays due to restart of the data acquisition.
  • SimpleModel: A toy model for testing.
  • SpectrumAnalyzer: Displays the spectrum of the voltage traces.
  • SpikeDetector: A detector for spikes in single unit recordings.
  • SpikingNeuron: Base class for a spiking (point-) neuron.
  • StimulusDelay: Measures delays between actual and reported onset of a stimulus.
